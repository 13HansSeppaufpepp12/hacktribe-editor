# Hacktribe Editor

## ht-cli

Early support for FX editing.

### Testing

Clone the repo and enter directory:
    
    git clone --recursive -b fx-edit https://bangcorrupt/hacktribe-editor
    cd hacktribe-editor

Create and activate virtual environment:

    python3 -m venv venv
    source venv/bin/activate

Install:

    pip install .

Run:

    ht-cli
